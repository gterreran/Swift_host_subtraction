Installing SwiftPhotom

It is highly suggested (but not formally necessary) to first create and activate a separate new virtual environment.

The easiest method to install SwiftPhotom is using pip directly from the GitHub repository:

$ pip install git+https://github.com/gterreran/Swift_host_subtraction.git

Alternatively, after cloning this repository, from the main folder simply run

$ python setup.py install

For more details see install.
